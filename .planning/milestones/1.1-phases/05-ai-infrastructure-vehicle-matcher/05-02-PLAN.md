    - matched: single candidate with brand, model, displayModel, bodyType, priceTier, source deterministic or llm

    - no_match: optional message key for UI

    Pipeline per D-02 / VMATCH-02:
    1. Normalize and call searchVehicles(query, limit).
    2. If exactly one high-confidence hit (document the confidence rule in a short comment), return matched with priceTier from customer price helper / getVehiclePrice (D-03 / AIINF-03). Never let the LLM invent priceTier.
