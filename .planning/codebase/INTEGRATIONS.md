# Integrations

## Core Integrations
1. **Convex**:
   - Backend as a Service for database state and real-time syncing.
   - Configured via `convex/` folder.

2. **Vercel Analytics**:
   - Page view tracking and user telemetry out-of-the-box.
   - Initialized globally in the Next.js App Router setup.

3. **Vercel Deployment**:
   - Hosting and serverless function execution.
   - Managed via `vercel.json` for custom configurations.
