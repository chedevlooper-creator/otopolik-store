# Tech Stack

## Overview
This project uses a modern React stack optimized for edge deployments and realtime data synchronization.

## Languages
- **TypeScript** (v5)
- **Node.js** (v20)
- **JavaScript** (ES6+)

## Core Frameworks
- **Next.js** (16.2.10) - App Router, React Server Components (RSC)
- **React** (19.2.4)
- **Tailwind CSS** (v4) - Styling and layouts

## Backend & State
- **Convex** (^1.42.1) - Real-time database and serverless functions

## Key Libraries
- **Lucide React** - Iconography
- **Lenis** - Smooth scrolling (`lenis` ^1.3.25)
- **ESLint** (^9) - Code linting

## Infrastructure
- **Vercel Analytics** (`@vercel/analytics`) - Telemetry
- Vercel Deployment configured (`vercel.json`)
