# Phase 9 ΓÇË UI Design Contract

> Visual and interaction contract for frontend phases. Generated by gsd-ui-researcher, verified by gsd-ui-checker.

**Phase framing:** Phase 9 is a storefront **boundary/wiring** phase (BOUND-01ΓÇô04), not a
visual redesign phase. It does not introduce new visual components, tokens, or layouts ΓÇË it
gates existing chrome (Header/Footer nav, `/destek` fallback, configurator AI mount) behind a
new flag and trims dead links. This contract documents the existing design system as the
constraint envelope for that wiring, and locks the copy/layout contract for the one surface
whose content actually changes for an end user: the `/destek` non-AI fallback page. Homepage
visual/motion changes are explicitly out of scope (Phase 10) ΓÇË see `09-CONTEXT.md` deferred list.
